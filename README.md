# RetailIQ — Adwuma Pa Intelligence Platform

> Smart Retail Management for Ghanaian MSMEs
> Built by Center for Analytics Services (CAS) · Backed by Akofena Capital Partners

## Features
- **Dashboard** — Live KPIs, revenue vs expenses charts, top products, category breakdown
- **Sales Tracking** — Record sales with MoMo/Cash/Card/Credit payment methods
- **Inventory Management** — Full CRUD, low-stock alerts, margin tracking
- **Reports & P&L** — Profit and Loss with period filtering and 6 chart types
- **Customer Management** — Profiles, purchase history, loyalty points, VIP status
- **Supplier Directory** — Contacts, ratings, order history
- **Smart Alerts** — Stock warnings, insights, achievements, credit eligibility
- **Business Health Score** — 0-100 composite score qualifying users for working capital loans
- **Settings** — Business profile editor, credit checker

## Quick Start
```bash
git clone https://github.com/YOUR_USERNAME/retailiq.git
cd retailiq
npm install
npm start
```

## Tech Stack
- React 18 + TypeScript
- Tailwind CSS 3
- Recharts (charts)
- React Router DOM v6
- Lucide React (icons)
- localStorage (offline-first)
